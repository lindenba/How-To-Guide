The category Effects provides animation to the web page. Simple basic effects are hide, show, and toggle. Hide and show methods are pretty obvious. Hide hides the element indicated and show shows the element indicated. Toggle is pretty neat in that whatever selector is called will fade away. This can be done fast or slow. Walking through the test code below 'p' is the selector '.show/hide/toggle' is the method and the parentheses is where you put your argument. So for toggle I put slow inside the parathesis. I wanted 'item 3' to slowly disappear after the webpage loads. You can also see that I gave the first 3 'p' elements either a class or an id so to be able to show, hide, and toggle. Similar to CSS a class gets a period when called and a pound sign is used for an id. 

$(document).ready(function()
    {
      $('p.a').show();
      $('p#b').hide();
      $('p#c').toggle('slow');

    });
    </script>
    <p class="a">item 1</p>
    <p id="b">item 2</p>
    <p id="c">item 3</p>
    <p>item 4</p>
    <p>item 5</p>
    <p>item 6</p>

Another example that I find to be very interesting is the .animate() method. The animate method is a custom method which performs a set of CSS properites. The code below is a circle moving from the right of the screen. If you look at the code closely outside of the jQuery function I defined the circle with a radius of 200px. However in the jQuery code I made the circle radius having a radius of 60px. When this runs the circle looks like it is flying out of the screen onto the webpage because of the effect from large to small and moving right. Looks a little trippy.

<script type="text/Javascript">
$(document).ready(function()
{
  $("button").ready(function()
  {
    $("div").animate({right: '800px', height:'120px', width:'120px', opacity:'.4', borderRadius:'60px'});
  });

});
</script>
<div style="background: green; height:400px; width:400px; position: absolute; border-radius:200px">
    <h3 style="padding-left:40px">Watch!</h3>
</div>
