I have shown a few things that jQuery can do that is interesting. My next trick is to show how to display Google Maps on your webpage. There isn't much code to be used. Google Maps API is just a source request defined with script. Similar to the jQuery request. 

<script >
      $(document).ready(function()
      {
        var map;
      var coordinates = new google.maps.LatLng( 45.5200, -122.6819);	// Coordinates of portland, or


      var mapOptions = { zoom: 8, center: coordinates  };


      map = new google.maps.Map( document.getElementById('map'), mapOptions );
      });

      </script>
      <style>

      #map
      {
      width : 200px; 	/* map width */
      height: 200px;	/* map height */
      }
      </style>
      </head>

      <body>
        <h2>Google Map</h2>
      <div id='map'>
      </div>

To add the google maps API request to get the image to load on your page. Just this url request that can go at the top of your HTML code. In the same place that you would request the jQuery.

<script src="https://maps.googleapis.com/maps/api/js?v=3.exp&sensor=false"></script>
