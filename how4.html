This last bit I'll go over an AJAX call using a post. AJAX stands for Asynchronous Javascript and Xml. This request from the website can send data to and retrieve from a server in the background. This happens without interfering with the page and what is already displayed.

What I am going to do is show a Post request to the server that will return the information inputted. First I define a text input box and a submit button. Which I do within the body part of the HTML. 

<body>
<h4>AJAX </h4>
<div class="entry-wrapper">
  <input type="text" size="30" id="input"/>
  <input type="button" value="submit" id="button"/>
</div>
<div class="response">
  <textarea rows="5" cols="30" id="responsetext"></textarea>
</div>
</body>

Next I make my document request jQuery. Where I define the button element to be useable. Within this function make the request to the server using .post request. I am using for my url a predefined .php which will hold the request and then echo back what is typed in.

$(document).ready(function()
{
  $('#button').click(function(){
    $.post("ajax.php",

    {
      dog_type:$('#input').val()
    },

    function(data)
    {
      $('#responsetext').val(data);
    });
  });

});

This code is what goes into the .php file.

<?php

$dog_type=$_POST['dog_type'];

echo"$dog_type is a type of dog";

?>

All together the code will output something like this. Where you can input a word and something will return. I set it up that if you type in a type of dog the response will be " is a type of dog".
