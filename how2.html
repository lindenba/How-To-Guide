Selectors in jQuery allow you to manipulate HTML elements by selection. Basically what the term means to select items. The selectors are used to select HTML elements by there id, class, type, attributes, and so on. There are a ton of ways to manipulate those elements.

The code below will specifically select all odd valued items. Any 'p' element that is odd will get a background of pink. Can also select anything greater than a certain number such as anything greater than 3 will be pink. Instead of 'p:odd' it can be 'p:gt(3)' and if you wanted to look at a specific value equal to some number this can be used 'p:eq(1)'. For the same code below you can use the element methods I went through in the Elements section. For example use .hide() or .fadeOut() which is another effects method.

<script type="text/Javascript">
$(document).ready(function()
{
  $('p:odd').css("background-color", "pink");
});
</script>
<p class="a">item 1</p>
<p id="b">item 2</p>
<p id="c">item 3</p>
<p>item 4</p>
<p>item 5</p>
<p>item 6</p>

More examples of HTML elements being manipulated. In this code below any element that has 'some words' will slowly disappear. With the .append will add 'and more words' will be added to what has been defined. I also had commented out .css, which would have changed the colors of all that had 'some words' defined.

$(document).ready(function()
        {
          var $word=$("p:contains('some words')");//.css("color", "yellow");
        //  $word.hide();
        $word.each(verb);
        function verb()
        {
          $(this).toggle("slow");
          $(this).append(" and more words");
        }
        });
        </script>
        <p class="a">item 1 some words</p>
        <p id="b">item 2 some words</p>
        <p id="c">item 3</p>
        <p>item 4</p>
        <p>item 5 some words</p>
        <p>item 6</p>
