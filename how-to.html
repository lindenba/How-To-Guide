This is my how to guide on jQuery. I am pretty new to web development and mostly been plugging away at HTML, Css, JavaScript, and PHP. All these different names for web development can get a little confusing. I never know what is what really. Which brings me to jQuery. I have heard about jQuery and thought it was some type of program to run code for websites. Yes jQuery is code for websites, but it is a JavaScript library. What does that mean exactly? From wikipedia "A JavaScript library is a library of pre-written JavaScript which allows for easier development of JavaScript-based applications, especially for AJAX and other web-centric tecnologies" That explains it right? For me, it actually does when you get down to implementing it into a system. JQuery is pretty cool and makes JavaScript much more interesting and neat. This how to guide will explore some aspects of jQuery.

Let's start at the beginning and talk about how to implement jQuery in your JavaScript code. JQuery doesn't need to be downloaded it can, but isn't necessary. All that is needed is to use jQuery with a CDN, which stands for content delivery network. This network hosts jQuery on servers across the world. This allows little re-downloading because the information is accessable anywhere.

How this is done is very simple. In your HTML reference the file using a script tag and paste into the header tag.

             <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>

After referencing jQuery using a CDN it is now possible to get started coding. With jQuery we are going to to make the webpage now perform some magic.

JQuery has a basic syntax, it uses a document ready function. When the webpage loads the code in this function does also. Instead of after the page has rendered, which happens with JavaScript. Example of the basic syntax can be found below.

$(document).ready(function()
          {
          //code goes here
          });
